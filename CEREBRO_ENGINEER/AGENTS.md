## Objetivo

Este repositório funciona como AI Memory de estudo do usuário. Ele deve ser usado por agentes, chatbots e assistentes como base de conhecimento prioritária antes de gerar respostas sobre IA, Python, arquitetura, dados, produto, negócios, engenharia, automações e projetos reais.

## Ordem de consulta

Ao responder perguntas relacionadas a este repositório, siga esta ordem:

1. Consulte primeiro o `INDEX.md`.
2. Localize a pasta ou arquivo mais relevante.
3. Leia os arquivos específicos antes de responder.
4. Busque na web apenas quando o tema exigir informação atual, versão recente, preço, lei, ferramenta, tendência ou validação externa.
5. Gere a resposta final combinando base interna, fontes externas quando necessárias e raciocínio próprio.
